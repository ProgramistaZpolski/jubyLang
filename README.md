# jubyLang
język programowania podobny do ruby. Ale tylko w syntaxie. W komendach nie.
### Komendy
```
badosz [tekst]
```
Wypisuje na ekranie tekst
Przykład
```
badosz Hello World!
badosz 22
```
Odpowiednik w Ruby
```
puts "tekst"
```

```
java = [nowa wartość]
```
Ustawia zmienną java na nową wartość
Przykład
```
java = 10
java = smieci
```
Odpowiednik w Ruby
```
java = [wartość]
```

```
java
```
wypisuje zmienną java
Odpowiednik w Ruby
```
puts java
```
